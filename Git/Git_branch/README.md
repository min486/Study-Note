<h2 align='middle'>깃 브랜치 명령어</h2>

<p align='middle'>branch 생성 및 제거, 확인 등의 기능을 하는 명령어</p>

<br>
<br>

## 🔧내용

### 브랜치 생성&전환

```javascript
git checkout -b [브랜치명]
```



### 브랜치 전환

```javascript
git checkout [브랜치명]
```



### 브랜치 이름 변경

```javascript
git branch -m [기존 브랜치명] [변경할 브랜치명]
```



### 브랜치 확인

```javascript
git branch
```



### 브랜치 삭제

```javascript
git branch -d [브랜치명]
```
