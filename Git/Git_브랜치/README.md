## 🔧세부 내용

### 변수

```javascript
name = 'Min'
age = 30
```



### 타입

- let

  > 변할 수 있는 값 선언할 때

- const

  > 변하지 않는 값 선언할 때



### 자료형

- 문자형

  👉 " " or ' ' 사용 가능

  👉 ``(백틱)은 변수나 표현식도 넣을 수 있다

  ```javascript
  `My name is ${name}`
  `나는 ${20+1}살 입니다`
  ```
